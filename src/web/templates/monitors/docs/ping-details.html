ICMP Ping

Overview

The ICMP Ping health check can be used to validate the status of a server by performing an ICMP Ping request.

Monitor Details

This monitor will attempt to send one ICMP Ping request per run, if the request is replied to the monitor will return true. If for any reason the request fails the monitor will return false.

Host

This field can be either an IP Address or Hostname. The IP or Hostname supplied must be resolvable and addressable via the internet.

Broadcast pings are not allowed and will always return false.
