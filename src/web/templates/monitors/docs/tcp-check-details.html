TCP Port Check

Overview

The TCP Port monitor can be used to check the status of any TCP Port.

Monitor Details

This monitor will attempt to establish a TCP connection to the specified IP and Port. If the connection is established this monitor will return True, if the connection fails the monitor will return False.

Host

This field can be either an IP Address or Domain. The IP or Domain supplied must be resolvable and addressable via the internet.

Port

This field can be a port between the range of 1 and 65535
