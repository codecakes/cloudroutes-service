Datadog Alerts: Webhook Integration

Overview

Within Datadog you can setup an integration for webhook requests. The Datadog Alerts: Webhook Integration monitor is designed to allow you to integrate Datadog metric alerts with Runbook.

This integration allows you to add Runbook Reactions to your existing monitoring solution.

This monitor is very similar to our general purpose API. For this monitor you will require two key pieces of information, the monitors unique url and check_key.

The url is simply an address to send requests to. This address is unique to each monitor and should not be reused.

The check_key is essentially a unique key generated for each monitor. This key is unique and should be treated the same as any API Key, as it is used in conjunction with the unique url to authorize the request.

Custom Payload

When using Datadog's webhook integration you will need to specify a custom payload. This payload must include the check_key and title attributes at minimum. Additional details can be included however they are not used for processing the monitor data.

Minimum Custom Payload
{ 
"check_key": "{{ apikey }}",
"title" : "$EVENT_TITLE"
}
